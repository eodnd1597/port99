GSAP

웹 브라우저에서 애니메이션을 구현하기 위한 자바스크립트 라이브러리이다. 기존 CSS Animation이나 jQuery Animation보다 탁월한 퍼포먼스를 발휘할 수 있도록 최적화된 애니메이션 전용 라이브러리이다.

gsap.to(".box1 .circle7", 2, {left: "90%", scale: "2"});
gsap.to(".box1 .circle8", 3, {left: "90%", skewX: "60deg"});
gsap.to(".box1 .circle9", 3, {left: "90%", skewY: "60deg"});
});
